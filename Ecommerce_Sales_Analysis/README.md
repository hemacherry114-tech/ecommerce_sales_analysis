# E-commerce Sales Analysis

## Project Overview
This project focuses on analyzing an e-commerce sales dataset to find key business insights, clean data, and create a comprehensive dashboard.

## Methodology
- **Data Cleaning in MySQL**: Performed data cleaning, handling missing values (NULLs, 'NA'), and ensuring data types were correct.
- **Feature Engineering**: Created new columns for total revenue, order year, and order month to enable time-series analysis.
- **EDA in Python**: Used the pandas library to perform exploratory data analysis and validate the SQL findings.
- **Dashboard in Power BI**: Created an interactive dashboard with key visualizations and metrics.

## Key Insights
- Total sales and average order value.
- Top-selling products and top customers.
- Sales trends over time and by country.
